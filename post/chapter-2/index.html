A Recommendation System Based on Cosine Similarity

January 10, 2024

Problema

Este projeto pretende resolver um problema que é descrito a seguir. Uma start-up do ramo imobiliária tem como modelo de negocio a venda de quartos de apartamentos. A empresa tem um registro de 12000 proprietários de imóveis que estão interessados na compra dos quartos. O problema consiste em que há a possibilidade em que vários proprietários de perfis pessoais diferentes comprem quartos do mesmo apartamento. Isto poderia gerar conflito entre os proprietários e com a empresa por não realizar um estudo de compatibilidade entre os futuros vizinhos.

Solução

A solução que este projeto propõe consiste em um sistema de recomendação que quantifique, por técnicas de machine learning e de análise de dados, o nível de compatibilidade entre os possíveis proprietários do mesmo apartamento.

Para isto é necessário que os 12000 proprietários compartilhem algumas informações sobre suas preferências pessoais e de estilo de vida. Eles devem preencher um questionário com 17 perguntas. Algumas delas são:

  • Qual é o seu horário de trabalho típico?
  • Você acorda cedo ou é uma pessoa noturna?
  • Você prefere um lugar animado do que tranquilo?
  • Você prefere viver com animais de estimação?
  • Você recebe visitantes com frequência?

Com as respostas do questionário é possível criar un banco de dados de onde será calculado a compatibilidade dos proprietários. Para tanto, nós utilizamos a similaridade por cosseno. Visando utilizar essa técnica, consideramos como vetores a informação completa de um proprietário do banco de dados.

Utilizamos o aplicativo “streamlit” e a linguagem python para melhorar visualização dos resultados para os participantes na toma de decisões na empresa. Por meio dessa app criamos um formulário que solicita como input os IDs dos proprietários de um apartamento dado e também a quantidade de possíveis vizinho que compartilham similaridade com os outros proprietários. Por exemplo, na Figura 1 de baixo mostramos o caso em que estamos interessados em conhecer a similaridade de 12 vizinhos com os proprietários identificados, respectivamente, com os IDs 10, 678 e 456.
